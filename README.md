# Laravel Docs VitePress

🌐 [laravel.0x123.com](https://laravel.0x123.com)

A VitePress-powered Laravel documentation site with community-maintained Chinese translations.

## Website Features

- Official Laravel documentation
- Community-maintained Chinese translations
- Laravel open source changelogs
- Multiple documentation versions
- Full-text search
- Seamless language and version switching
- laravel.com-compatible URLs
- Content is synchronized with upstream sources and updated in near real time
